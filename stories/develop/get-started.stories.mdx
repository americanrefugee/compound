# Getting started as a developer

Find the documentation and issue tracker in [vector-im/compound](https://github.com/vector-im/compound) on GitHub.

## Projects

### [compound-design-tokens](https://github.com/vector-im/compound-design-tokens)

This project hosts a collection of design tokens that are transformed using style dictionary
so they can be consumed by all supported platforms.

The tokens are synchronised with Figma using [tokens.studio](https://tokens.studio/)

See the transformed assets for

- [Android](https://github.com/vector-im/compound-design-tokens/tree/develop/assets/android)
- [iOS](https://github.com/vector-im/compound-design-tokens/tree/develop/assets/ios/swift)
- [Web](https://github.com/vector-im/compound-design-tokens/tree/develop/assets/web)

## Feedback

If you spotted an issue, think of an enhancement, or spotted that something is missing,
we would love to hear about it. You can create an issue on the [GitHub Compound repository](https://github.com/vector-im/compound/issues/new)
or interact with the team in the internal [Compound room](https://matrix.to/#/#compound:matrix.org) on Matrix.
